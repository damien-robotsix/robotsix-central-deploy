Add `POST /chat/github/repos` so the chat agent can create GitHub repositories (e.g. `robotsix-chat-mobile`), reusing the same GitHub-App-installation account as the read-only Actions-status endpoints. Repo creation uses a separate PAT (`github_repo_create_token`, shared with robotsix-mill's `forge_repo_create_token`) since GitHub App installation tokens cannot create repositories under a personal account.
