Registers a new `github` virtual chat-agent component: `GET /chat/github/repos/{owner}/{repo}/actions/runs` and `GET /chat/github/repos/{owner}/{repo}/actions/runs/{run_id}` let the chat agent check GitHub Actions workflow-run status (e.g. whether a merged PR's image build/deploy has finished) without ever holding GitHub credentials itself. Read-only; the deploy server mints a GitHub App installation token server-side via PyGithub, reusing the same GitHub App installation as robotsix-mill. Configured via new `github_app_id`/`github_app_private_key` settings (empty by default — the component 503s until an operator provisions them).
